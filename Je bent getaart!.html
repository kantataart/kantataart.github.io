Wat is taarten?

Taarten is een traditie waarbij als iemand vergeten is zijn computer te vergrendelen, via de mail op deze computer aan de hele afdeling taart wordt beloofd.
Jouw collega heeft je gespaard, maar we willen wel de risico's benadrukken.

Waarom taarten?

Een onvergrendelde computer kan de aanleiding zijn van een security incident zoals:
  • Ongewenste toegang tot vertrouwelijke gegevens door gebruik te maken van jouw rechten (datalek)
  • Iemand kan zich voordoen als jou om anderen te misleiden, bijvoorbeeld via de mail
  • Iemand kan uit jouw naam ongewenst gedrag vertonen
Voorkom dit en vergrendel je computer dus altijd! Ook als je slechts kort van je plek bent. Wen jezelf bijvoorbeeld de volgende sneltoetsen aan:
  • Windows: Windows Toets + L
  • Mac: CTRL + CMD + Q
Bij vragen over informatiebeveiliging kan je altijd contact opnemen met je IT afdeling of met ons.
